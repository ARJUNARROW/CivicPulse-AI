# CivicPulse AI

## AI-Powered Emergency Incident Management using UiPath Maestro and AI Agents

CivicPulse AI is an end-to-end emergency incident management platform built on **UiPath Automation Cloud**. The solution combines **UiPath Maestro**, **AI Agents**, **Data Fabric**, **Integration Service**, **RPA Workflows**, and a **Node.js backend** to automate the complete incident response lifecycle.

### Key Features

- Incident Review & Verification
- AI Risk Assessment
- AI-based SLA Assignment
- AI-driven Priority Classification
- Intelligent Multi-Department Dispatch
- Human-in-the-Loop Escalation & Approvals
- Data Fabric Integration
- AI Incident Closure Report Generation
- Automated PDF Report Generation
- Email Notifications
