# CivicPulse AI - UiPath Solution

## Overview

This folder contains the UiPath implementation of CivicPulse AI built on UiPath Automation Cloud.

The solution is orchestrated using UiPath Maestro and combines AI Agents, Human-in-the-Loop tasks, Data Fabric, Integration Service, and RPA workflows to automate the complete emergency incident lifecycle.

## Major Components

- UiPath Maestro BPMN Process
- AI Agents
- UiPath Apps Forms
- Human Approval Tasks
- Data Fabric Entities
- RPA Workflows
- Integration Service
- Storage Buckets

## Workflow

The solution automates the following process:

1. Incident Review
2. AI Risk Assessment
3. SLA Assignment
4. Priority Classification
5. Department Dispatch
6. Dispatch Confirmation
7. Resolution
8. Data Fabric Update
9. AI Incident Closure Report
10. Report Generation
11. Final Email Notification

## Technologies

- UiPath Maestro
- UiPath Apps
- UiPath AI Agents
- UiPath Data Fabric
- UiPath Integration Service
- UiPath Storage Buckets
- UiPath Orchestrator
- UiPath RPA

## Deployment

Import the UiPath solution into UiPath Automation Cloud and configure the required connections, Data Fabric entities, AI Agents, Storage Buckets, and Integration Service before execution.
